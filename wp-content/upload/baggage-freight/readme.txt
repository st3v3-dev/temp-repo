== Installation ==

You can install this plugin directly from your WordPress dashboard:

 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *WP Crontrol*.
 3. Click *Install Now* next to the *WP Crontrol* plugin.
 4. Activate the plugin.

Alternatively, see the guide to [Manually Installing Plugins](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Usage =

1. Go to Baggage Freight Shipping -> Registration to setup your storeowner account. Your account will be created autometically on http://www.baggagefreight.com.au . 
   You can also create account on http://www.baggagefreight.com.au and put email and password in this plugin on registration section.
2. Go to Baggage Freight Shipping -> Settings to setup handling fee and freight calculation.
3. Go to Baggage Freight Shipping -> Default Freight Settings to setup handling fee and freight calculation.
4. Go to Baggage Freight Shipping -> Product Freight Values to put package details. You can upload csv file for package info.
[Example csv :
  SKU,Length,Width,Height,Weight
  PRO1,10,10,10,10 /n
  PRO2,20,20,20,20 /n
  PRO3,20,20,25,25
]

5. Go to Baggage Freight Shipping -> Payment Option to determine payment option on baggagefreight.
6. Go to Baggage Freight Shipping -> Shipping Label to go to your store-owner dashboard on baggagefreight.
6. Go to Baggage Freight Shipping -> Consolidate Products List to choose consolidate products.
